# 개발 도구

이 디렉토리에는 개발 및 분석 도구가 포함되어 있습니다.

## 파일

- `compare_trades.py`: 레거시와 신규 백테스트 엔진 간 거래 내역 비교
- `compare_metrics.py`: 서로 다른 백테스트 간 성능 지표 비교
- `check_logic_errors.py`: 백테스트 구현의 로직 오류 확인
- `verify_strategy_logic.py`: 전략 로직 일관성 검증
- `verify_strategy_logic_detailed.py`: 상세 전략 로직 검증

이들은 개발 도구이며 메인 CLI의 일부가 아닙니다.
